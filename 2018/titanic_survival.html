We will drop the "Cabin" feature because there are so many null values. In addition, 'Name', 'Ticket' and 'PassengerId' can be dropped because they should not have an impact on the result.

To do this, we create a transformer, in Scikit-Learn style, that converts a Pandas Data Frame to a Numpy 2d array at the same time and that we can reuse:

In [5]:
from sklearn.base import BaseEstimator, TransformerMixin

class DataFrameSelector(BaseEstimator, TransformerMixin):
    def __init__(self, attribute_names):
        self.attribute_names = attribute_names
    def fit(self, X, y=None):
        return self
    def transform(self, X):
        return X[self.attribute_names].values

We can also look more closely at the "Fare" and "Pclass" attributes, to see if they both have the same correlation with survival rate and thus one could be dropped.

The ticket class does correlate with the fare paid (as can be seen from the graph below). However, we also see that the fare seems to have an impact on the survival rate within each ticket class (people who paid the lowest fare seem to have the lowest survival rate irrespective of what ticket class they bought). Therefore, it could be useful to keep both attributes.

Next we need to take care of: 1) filling in the missing values (especially in the "Age" attribute) 2) converting categorical text attributes in to numbers because machine learning algorithms prefer numbers over text 3) feature scaling numerical attributes because machine learning algorithms often don't work well on numerical attributes with very different scales

Let's put all this into a pipeline so we can automatically complete the process later:

In [8]:
from sklearn.pipeline import Pipeline
from sklearn.pipeline import FeatureUnion
from sklearn.preprocessing import Imputer
from sklearn.preprocessing import LabelEncoder
from sklearn.preprocessing import StandardScaler

# split titanic set into feature and target sets
def splitTitanicSet(titanic_set):
    y = titanic_set['Survived']
    X = titanic_set.drop('Survived', axis=1)
    return (X, y)

num_attribs = ['Age', 'SibSp', 'Parch', 'Fare'] # numerical attributes
cat_attribs = ['Pclass', 'Sex', 'Embarked'] # categorical attributes

num_pipeline = Pipeline([
    ('selector', DataFrameSelector(num_attribs)), # select needed numerical attributes
    ('imputer', Imputer(strategy='median')), # fill missing values with median
    ('std_scaler', StandardScaler()) # scale features with standardization
])

cat_pipeline = Pipeline([
    ('selector', DataFrameSelector(cat_attribs)), # select needed categorical attributes
])

full_pipeline = FeatureUnion(transformer_list=[
    ("num_pipeline", num_pipeline),
    ("cat_pipeline", cat_pipeline)
])

# split dataset to feature and target sets
X_raw, y = splitTitanicSet(train_set)

# LabelEncoder does not support pipelines so convert attributes with text classes to numbers here
label_encoder = LabelEncoder()
X_raw['Sex'] = label_encoder.fit_transform(X_raw['Sex'])
X_raw['Embarked'] = label_encoder.fit_transform(X_raw['Embarked'].astype(str))

# execute pipeline
X = full_pipeline.fit_transform(X_raw)
